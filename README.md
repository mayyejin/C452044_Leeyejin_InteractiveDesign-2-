# C452044_Leeyejin_InteractiveDesign-2-
단순히 글로만 존재했던 문자들은 누군가의 말과 몸짓을 통해 의미와 생명력을 가진다. 특히 공연예술이나 미디어에서의 텍스트는 모두 배우의 해석을 거쳐가게 되고, 각자만의 목소리, 몸짓, 표정 등에 따라 새로운 의미를 계속해서 생성해 나간다. 이번 프로젝트를 통해, 이러한 배우(사람)와 대본 텍스트 사이의 관계성과, 단지 재현만의 용도였던 텍스트가 비로소 의미를 갖게 되는 순간을 인터랙티브 작업으로 나타내고자 했다.  텍스트는 표현의 여지가 다양한 셰익스피어의 ‘햄릿’을 선택했고, 배우들의 연기 목소리, 나의 해석을 통해 도출한 몸짓, 참여자가 입력하는 목소리에 ‘햄릿’의 대사 텍스트가 반응하도록 했다.
